GoveKits

一个基于 Unity 的游戏开发工具包，提供丰富的游戏系统组件和工具类。

功能特性

🎮 核心系统

• 效果系统 (Effect System) - 灵活的游戏效果管理框架

• 状态机 (FSM) - 有限状态机实现，支持复杂的状态转换逻辑

• 定时器系统 (Timer System) - 高性能的游戏定时器管理

🌐 网络功能

• Web API 请求处理 - 优化的网络请求处理系统

• HTTP 客户端 - 简化的 HTTP 通信接口

🏷️ 标签系统

• Gameplay Tag 系统 - 游戏对象标签管理（已重构移除）

技术栈

• 主要语言: C# (97.6%)

• 辅助工具: Python (2.4%)

• 平台: Unity 游戏引擎

项目结构


GoveKits/
├── Assets/           # 游戏资源文件
├── Packages/         # Unity 包管理
├── ProjectSettings/  # Unity 项目设置
└── .vscode/         # VS Code 配置
